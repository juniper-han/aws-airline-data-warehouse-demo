# 技术组件说明文档
## 实时消息组件
### AWS MSK（Kafka）
负责高并发实时数据接入，实现流量削峰、系统解耦、增量数据异步传输，是实时链路核心中间件。

## 数据湖与治理
### S3
企业级海量文件存储，作为统一数据湖承载离线日志与 Kafka 落地半结构化数据。
### Lake Formation
统一数据湖权限、元数据、访问策略管理，标准化企业数据治理流程。

## ETL 数据集成
### AWS Glue
全托管分布式 ETL 服务，负责数据抽取、清洗、转换，打通 S3 与 Redshift 数据流转。

## 数仓与KV存储
### Redshift
云端企业级数仓，承载分层建模、复杂SQL分析、指标计算，作为数据分析核心出口。
### DynamoDB
高性能键值数据库，用于维度字典、配置类数据高速查询。

## 自动化与调度
### Lambda
无服务事件驱动计算，实现 Kafka 消费、定时触发、轻量自动化任务。
### Step Functions
可视化工作流编排，串联多类AWS任务，实现流程自动化、失败重试。

## 监控告警
### CloudWatch
日志、指标、链路延迟全链路观测，保障数据链路稳定性。
### SNS
统一消息推送，针对任务失败、数据延迟、服务异常实时告警。
