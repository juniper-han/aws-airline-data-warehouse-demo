# AWS Airline Data Warehouse Demo
## Project Introduction
A cloud-native stream-batch integrated data solution built with AWS services and Apache Kafka (AWS MSK).
Covers real-time data ingestion, data lake storage, ETL integration, data warehouse modeling, workflow orchestration, monitoring and alerting.

## Technology Stack
### AWS Services
- Amazon Redshift
- Amazon Glue
- AWS MSK (Kafka)
- AWS Lambda
- AWS Step Functions
- Amazon S3
- Lake Formation
- Amazon DynamoDB
- Amazon CloudWatch
- Amazon SNS

### Big Data Technology
- Apache Kafka real-time streaming
- Data warehouse layered modeling
- ETL data integration & data cleansing
- Stream-batch unified data processing

## Core Features
1. Real-time data collection and consumption based on Kafka
2. Data lake permission and metadata governance
3. Automated ETL processing and data transformation
4. Standard layered data warehouse modeling
5. End-to-end workflow scheduling & exception alerting
